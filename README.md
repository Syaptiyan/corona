# 🦠 COVID-19 Indonesia

Dashboard real-time data COVID-19 di Indonesia dengan visualisasi interaktif dan data lengkap.

## ✨ Fitur

- 📊 **Statistik Real-time** - Kasus, Kematian, Sembuh, Aktif
- 📈 **Grafik Interaktif** - Kumulatif & Harian dengan filter waktu
- 🥧 **Pie Chart** - Komposisi kasus Indonesia
- 📉 **Mortality & Recovery Rate** - Persentase kematian & kesembuhan
- 🌏 **Perbandingan ASEAN** - Indonesia vs Malaysia, Singapura, Thailand, Vietnam, Filipina
- 🏆 **Milestone** - Timeline pencapaian kasus
- 💡 **Tips Kesehatan** - Pencegahan COVID-19
- 📱 **Share WhatsApp** - Bagikan data ke WhatsApp
- 📥 **Export CSV** - Download data lengkap
- 🔄 **Auto-refresh** - Update otomatis setiap 5 menit

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- [Chart.js](https://www.chartjs.org/) - Library grafik
- [disease.sh API](https://disease.sh/) - Sumber data COVID-19
